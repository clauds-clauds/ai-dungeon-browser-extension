Share / Import Data

Export

This will package all characters and notes for the current adventure into a single `.json` file that you can save as a backup or share with others.

Import & Merge

Important: This will try to merge all data from another `.json` file. Generally the safest way to import data from other adventures.

Import & Replace

Warning: This will overwrite all existing characters and notes for the current adventure. This action cannot be undone.
